## Abstrak (Bahasa Indonesia)

**RANCANG BANGUN SISTEM INFORMASI ABSENSI GURU MENGGUNAKAN QR CODE DAN PENILAIAN SISWA BERBASIS WEB DENGAN METODE SAW (STUDI KASUS: SMPN 4 PURWAKARTA)**

SMPN 4 Purwakarta masih menggunakan sistem pencatatan kehadiran guru secara manual melalui buku absensi yang menimbulkan berbagai kendala seperti ketidakakuratan data, proses rekapitulasi yang lambat, serta tingkat pengisian absensi yang fluktuatif dengan 32–40% guru tidak mengisi absensi setiap bulannya. Selain itu, proses penilaian dan pemeringkatan siswa juga dilakukan secara manual menggunakan Microsoft Excel tanpa standar baku, sehingga membutuhkan waktu 1–2 minggu per semester dan kurang objektif karena hanya berdasarkan nilai rata-rata tanpa mempertimbangkan bobot setiap aspek penilaian. Penelitian ini bertujuan untuk membangun sistem informasi berbasis web yang mengintegrasikan absensi guru menggunakan QR Code dengan validasi geolokasi dan penilaian siswa menggunakan metode Simple Additive Weighting (SAW) dalam satu platform terpadu. Sistem dikembangkan menggunakan metode Prototype dengan framework Laravel 12 dan basis data MySQL. Pengujian dilakukan melalui Black Box Testing (58 skenario, 100% berhasil), White Box Testing menggunakan Basis Path Testing (19 jalur independen, 100% lolos), serta User Acceptance Testing (UAT) yang melibatkan 34 peserta. Hasil UAT menunjukkan skor rata-rata 4,46 dari 5,00 (89,29%) dengan kategori "Sangat Baik" pada seluruh aspek yang dievaluasi, yaitu fungsionalitas (90,59%), kemudahan penggunaan (88,24%), kecepatan sistem (90,00%), tampilan antarmuka (86,47%), dan keandalan (91,18%). Implementasi sistem berhasil mempercepat proses presensi dari sekitar 2 menit menjadi kurang dari 10 detik per guru, mengubah proses rekapitulasi yang semula memerlukan 2–3 hari menjadi instan melalui fitur export otomatis, serta menghasilkan pemeringkatan siswa yang objektif dan transparan berdasarkan empat kriteria berbobot. Seluruh responden menyatakan bahwa sistem layak digunakan secara operasional di lingkungan sekolah.

**Kata Kunci:** Sistem Informasi, Absensi Guru, QR Code, Geolokasi, Penilaian Siswa, Simple Additive Weighting (SAW), Laravel, Prototype, SMPN 4 Purwakarta

## Abstract (English)

**DESIGN AND DEVELOPMENT OF A TEACHER ATTENDANCE INFORMATION SYSTEM USING QR CODE AND WEB-BASED STUDENT ASSESSMENT WITH SAW METHOD (CASE STUDY: SMPN 4 PURWAKARTA)**

<sup>1</sup>Muhammad Rifky Akbar, <sup>2</sup>Garno, <sup>3</sup>Betha Nurina Sari

<sup>1</sup>2210631170088@student.unsika.ac.id <sup>2</sup>Garno@staff.unsika.ac.id <sup>3</sup>betha.nurina@staff.unsika.ac.id

Department of Informatics Engineering
Faculty of Computer Science, Universitas Singaperbangsa Karawang

SMPN 4 Purwakarta still relies on a manual attendance recording system using physical logbooks, which causes several issues including data inaccuracy, slow recapitulation processes, and fluctuating attendance completion rates with 32–40% of teachers failing to record their attendance each month. Additionally, student assessment and ranking processes are conducted manually using Microsoft Excel without standardized procedures, requiring 1–2 weeks per semester and lacking objectivity as rankings are based solely on average scores without considering the weight of each assessment criterion. This study aims to develop a web-based information system that integrates teacher attendance using QR Code with geolocation validation and student assessment using the Simple Additive Weighting (SAW) method within a unified platform. The system was developed using the Prototype method with the Laravel 12 framework and MySQL database. Testing was conducted through Black Box Testing (58 scenarios, 100% passed), White Box Testing using Basis Path Testing (19 independent paths, 100% passed), and User Acceptance Testing (UAT) involving 34 participants. The UAT results showed an average score of 4.46 out of 5.00 (89.29%), categorized as "Very Good" across all evaluated aspects: functionality (90.59%), ease of use (88.24%), system speed (90.00%), user interface (86.47%), and reliability (91.18%). The system implementation successfully reduced the attendance process from approximately 2 minutes to less than 10 seconds per teacher, transformed the recapitulation process from 2–3 working days to instant through automated export features, and produced objective and transparent student rankings based on four weighted criteria. All respondents agreed that the system is suitable for operational use in the school environment.

**Keywords:** Information System, Teacher Attendance, QR Code, Geolocation, Student Assessment, Simple Additive Weighting (SAW), Laravel, Prototype, SMPN 4 Purwakarta
